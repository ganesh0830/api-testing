# Newman Reporter - Junit Extended

A enhanced [Newman](https://github.com/postmanlabs/newman) reporter for Junit files.  This is a fork of the standard Junit that comes with Newman. Orignal author Postman Labs help@getpostman.com. The reporter adds more detailed failure information to the Junit files. 

## Requirements 
- [Node.js](https://nodejs.org/en/) => v6.11

## Dependencies
 #### Node.js packages
 - [lodash](https://www.npmjs.com/package/lodash)
 - [xmlbuilder](https://www.npmjs.com/package/xmlbuilder)

## Getting Started 
1. Install the [REST API Testing Framework](https://bitbucket.fsc.atos-services.net/projects/ED/repos/rest-api-testing-framework/browse)
2. To create a npm package file run the command ```npm pack``` within this project. 
3. Copy the npm package to the root of the REST API Testing Framework project directory.
4. Run the command ```npm install <tarball file path>``` to install the **Newman Reporter Junit Extended** package.
